come da conversazione telefonica le esplicito le voci quotate dal competitor  per la realizzazione di un sito web per una spa:

 

-          Studio layout grafico del sito internet e della UX e UI rispetto al target specifico, con analisi preventive fatte sui dati forniti dal cliente, se richiesto.

-          Realizzazione sito e-commerce con possibilità di prenotazione delle fasce orarie dei servizi e la gestione delle fatture eletroniche integrata nel sistema di amminastrazione admin.

-          Realizzazione in due lingue (traduzione fornita dal cliente) più costo parziale per ogni lingua aggiuntiva

-          Ottimizzazione del posizionamento rispetto ai dispositivi in uso con sistemma reattivo che possa addatarsi e essere manutenuto nel tempo con maggiore facilità

-          Testi e immagini fornite dal cliente e/o create tramite l'utilizzo di strumenti di IA

Il sito / web APP che proponiamo sarà una one page application con elementi reattivi quali, il formato pagina, le animazioni e l'interazione coi menu, proproniamo anche di studiare un UX che rispetti i parametri migliori rispetto al target la dove un analisi sia possibile, richiesta; al interno del sito sarà presente un sistema di registrazione e incapsulamento dati che permetta due tipi di utenze: Utenti normali che usano e pagano e prenotano i servizi e Utenti Admin che si occupano della gestione del Sito / Web APP e che possono usufuire di strumenti di calcolo e/o di fatturazione assistita. 

Il sito / Web APP sarà anche fonito di una modalità di esecuzione test che permetta un atuo check up delle parti del sistema (Unit and relational test  [Junit]  )

La traduzione che verrà fornita dal cliente sarà disponibile per 2 lingue senza costi aggiuntivi, e dovrà essere fornita dal cliente, le altre lingue che potranno essere aggiunte saranno discusse in fase di progettazione e potranno essere inserite tramite accordo fra le parti.

Proponiamo una manutenzione di 6 mesi da dopo il rilascio della 1.0 prograbbile di altri 6 in caso di accordo fra le parti, in caso di mancato accordo si verificherà uno scigliomento dei doveri di manutenzione ordinaria ai sensi del tacito accordo. 

Le tecnologie che vorremmo utilizzare sono: Angular, Node.js, React, MySql e Java ( puro )
